decrypt <filename>

Decrypts the given file or directory. 

Some files or directories must be decrypted before they can be accessed. When running this command, a prompt will appear, asking for a password. The correct password must be given before the file or directory will be decrypted.

Examples:

Decrypt a file called 'file.txt'

	decrypt file.txt

Decrypt a directory called 'files'

	decrypt files
